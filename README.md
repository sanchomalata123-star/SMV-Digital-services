# SMV Digital Services Website

A **10-page interactive website** showcasing SMV Digital Services’ offerings with modern layout, animations, and demo payment functionality. Fully built with **HTML, CSS, and JavaScript**, ready to deploy via **GitHub Pages**.

---

##  Features

- **10 fully designed pages**:  
  - Home, About, Services, Portfolio, Blog, Contact, FAQ, Testimonials, Pricing, Terms & Conditions  
- **Modern CSS Layout**:  
  - Gradient headers, card layouts, responsive design, hover effects, rounded buttons  
- **Interactive JavaScript**:  
  - Smooth scroll navigation  
  - 3D card hover tilt  
  - Dynamic counters for stats  
  - Typing text animations  
  - Modal popups  
  - Demo payment buttons with animations  
- **Responsive Design**: Works on desktop, tablet, and mobile devices  
- **GitHub Pages Ready**: Deploy your website in minutes
